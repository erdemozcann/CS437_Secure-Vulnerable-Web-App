# Secure & Vulnerable Web Application

This project demonstrates common web security vulnerabilities and their secure implementations. It was developed as part of the CS437 Software Security course.

## Overview

The application includes both vulnerable and secure versions of key web functionalities to illustrate how attacks are performed and how they can be mitigated in real-world systems.

## Vulnerabilities Covered

- Cross-Site Scripting (XSS)
- SQL Injection
- Cross-Site Request Forgery (CSRF)
- Server-Side Request Forgery (SSRF)
- Command Injection

## Features

- Side-by-side vulnerable and secure implementations
- Practical attack scenarios and exploitation examples
- Demonstrations of mitigation techniques and best practices
- Clear comparison between insecure and secure coding approaches

## Technologies Used

- php (Frontend & Backend)
- Docker (for containerization)
- Security testing and analysis tools
